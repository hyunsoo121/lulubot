# lulubot

디스코드 안에서 완결되는 롤 내전 생태계.
토너먼트 코드 발급 → 게임 종료 시 자동 전적 수집 → 디스코드에서 통계 조회까지, 디스코드를 벗어나지 않고 모든 게 가능한 봇 서비스.

---

## 기술 스택

| 분류 | 기술 |
|------|------|
| 언어 | TypeScript |
| 봇 | discord.js |
| 웹서버 | Express |
| ORM | Prisma |
| DB | PostgreSQL |
| 캐시 | Redis |
| 배포 | Docker + AWS |

---

## 서비스 플로우

```
/내전생성
  └─ 토너먼트 코드 발급 (Redis PENDING, TTL 30분)
       └─ 게임 종료 → Riot이 Express 서버로 matchId POST
            └─ Match API 호출 → DB 저장 → 참가자 매핑
                 └─ 결과 임베드 디스코드 채널 자동 출력
```

## 명령어

### 계정
| 명령어 | 설명 |
|--------|------|
| `/등록 닉네임#태그` | 라이엇 계정 연결 |
| `/내정보` | 연결된 계정 확인 |

### 내전
| 명령어 | 설명 |
|--------|------|
| `/내전생성` | 토너먼트 코드 발급 |

### 전적 조회
| 명령어 | 설명 |
|--------|------|
| `/전적 [@유저 or 닉네임]` | 이 서버에서의 통계 |
| `/전체전적 [@유저 or 닉네임]` | 전 서버 합산 커리어 |
| `/최근경기 [@유저]` | 마지막 내전 결과 상세 |

### 통계
| 명령어 | 설명 |
|--------|------|
| `/랭킹` | 서버 내 종합 랭킹 |
| `/라인랭킹 [라인]` | 라인별 랭킹 |
| `/챔피언 [챔피언명]` | 챔피언별 서버 통계 |
| `/듀오 @유저1 @유저2` | 두 유저의 케미스트리 |
| `/라이벌 [@유저]` | 나의 라이벌 확인 |

### 재미
| 명령어 | 설명 |
|--------|------|
| `/업적 [@유저]` | 달성한 업적 목록 |
| `/이달의기록` | 이번 달 각종 1위 |

### 관리자
| 명령어 | 설명 |
|--------|------|
| `/멤버등록 @유저 닉네임#태그` | 대신 등록 |
| `/데이터초기화` | 테스트용 초기화 |

## 개발 단계

- **1단계 (데모)** — 유저 등록, 과거 기록 스캔, 전적/랭킹 조회, Production Key 심사 신청
- **2단계 (정식)** — 토너먼트 코드 발급, Riot 콜백 자동 수집, 결과 자동 출력
- **3단계 (확장)** — 업적 시스템, 듀오/라이벌, 칭호, 이달의기록

## 로컬 개발 환경

### 사전 준비

- Node.js 20+
- Docker & Docker Compose
- Riot Games API Key

### 설치

```bash
# 의존성 설치
npm install

# Git 훅 설치
sh hooks/install.sh

# 환경변수 설정
cp .env.example .env

# DB + Redis 실행
docker-compose up -d

# Prisma 마이그레이션
npx prisma migrate dev

# 봇 실행
npm run dev
```

### 환경변수

`.env.example` 참고.
